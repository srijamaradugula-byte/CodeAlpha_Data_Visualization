 Data Visualization Project

Project Description
This project performs data visualization using Python on the built-in Iris dataset. It demonstrates various charts and plots to understand data distribution, relationships, and patterns without using any external CSV file.

Features
- Load dataset using `scikit-learn` (no download required)
- Dataset overview and structure
- Histogram visualization
- Boxplot for outlier detection
- Correlation heatmap
- Pair plot for feature relationships
- Scatter plot between features
- Bar chart for class distribution
- Pie chart for category distribution
- Line chart for trend visualization

Dataset Used
- **Iris Dataset (Built-in from scikit-learn)**  
  - Features: Sepal length, sepal width, petal length, petal width  
  - Target: Species (Setosa, Versicolor, Virginica)

Technologies Used
- Python 3
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

Installation

Install required libraries:

pip install pandas matplotlib seaborn scikit-learn

Usage

Run the script:

python visualization.py

 Output
The program generates the following visualizations:
- Histogram
- Boxplot
- Correlation Heatmap
- Pair Plot
- Scatter Plot
- Bar Chart
- Pie Chart
- Line Chart
